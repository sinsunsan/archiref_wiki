### IE Compatibility 
```` html
 <!--[if lte IE 9]>
       <script
src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
<![endif]-->
````
This code use conditionnal comment to target IE9 and lower 
http://www.quirksmode.org/css/condcom.html
